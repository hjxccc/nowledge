# 案例 · 深学模式带读外部真源 + 复训 loop 活样板

这个案例演示 nowledge 两个最核心、别处没样板证明的能力：

## ① 带读一个真实外部 repo（反 slop 的活证据）
学习对象是吴恩达《神经网络和深度学习》——用户自己 clone 的公开仓库
`github.com/fengdu78/deeplearning_ai_books`（黄海广团队整理，免费非商用）。

nowledge **不重写笔记**（黄老师写得比现编强、且会 slop），而是**带读那份真材料**，
只加大 V 没给的三件增值：① 该周重点/生词对齐 ② 笔记一句带过、但会踩的坑 ③ 记住 3 点。
→ 见 `ROADMAP.md`（例：Week2 把"为什么用交叉熵不用平方误差""学习率太大发散"这些
笔记轻描淡写的坑单拎出来讲透）。

## ② 复训 loop 真在转（差异化的活样板）
`progress.json` 不是静态占位，是**一次真实学习 session 后的状态**：
- 学习中现场出题回考，答不上的当场记成"洞"入队；
- `#2 交叉熵` 与 `#6 学习率` 都 `lapses=1`（真的答错过、被打回从头、近期高频重考）；
- `history` 里有真实的 `fail` 记录 + 到期日重排。

这就是 nowledge 和"读一遍点头 = 流畅性错觉"的分界：**洞不会被放过**。

## 脱敏说明
本目录**只含 nowledge 生成的外壳**（路线 + 复训队列 + 本说明），
**不含任何版权原文**——原文始终在用户本地 clone 的仓库里（"原文在外、外壳在本地"正是 skill 的模型）。

## 复现
```bash
git clone https://github.com/fengdu78/deeplearning_ai_books   # 自备原文
# 然后对着 agent 说："用 nowledge 带我学这个 repo 的第一门课前几周"
```
