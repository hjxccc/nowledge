Loop 的解剖：五件套 + 一个外置记忆

loop-engineering-nowledge · 图解 · 来源 Addy Osmani / Steinberger 清单（sources/）

Harness 单个 agent 运行的环境；loop 站在它上面一层
Loop 递归目标：定义目的，AI 迭代到完成，你不再逐轮 prompt
外置记忆 磁盘上的 md/看板，跨轮次存"已完成/下一步"

旧：prompt engineering

你握着工具，一轮一轮 prompt：打字→读结果→再打字。你始终在环里。

新：loop engineering

你设计一个会自己找活、派活、自查、记账、定下一步的系统，让它去 poke agent。你退到环外。
